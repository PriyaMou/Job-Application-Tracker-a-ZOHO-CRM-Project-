The Job Application Tracking System is a CRM-based project designed to manage and monitor the entire job application process in a structured and efficient way. This project was implemented using Zoho CRM to simulate a real-world recruitment tracking workflow.

This project provided hands-on experience in understanding how Customer Relationship Management (CRM) systems can be adapted for personal productivity and professional workflow management.
